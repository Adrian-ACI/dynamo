Historias de usuario


No. Rol Característica/funcionalidad Razón/Resultado
US0001 Como usuario Quiero iniciar sesión en el sistema. Con el fin de hacer uso de las funcionales a las que puedo acceder.
US0002 Como usuario Quiero publicar reseñas de libros cuando estoy logueado en el sistema. Con el fin de que otro usuario pueda leer mi reseña, hacerle comentarios y calificarla.
US0003 Como usuario Quiero leer trabajos de otros autores. Con el fin de poder reseñarlas o basarse en ellas para futuras obras.
US0004 Como usuario Quiero publicar mis obras literarias. Con el fin de que otras personas puedan leer mis obras.
US0005 Como usuario Quiero registrarme en la plataforma del sistema. Con el fin de hacer uso de las funcionales a las que puedo acceder.
US0006 Como usuario Quiero poder elegir un género literario para leer o reseñar. Con el fin de poder reseñar o leer libros del género literario de mi agrado.
US0007 Como usuario Quiero poder guardar mis libros o reseñas favoritas. Con el fin de poder leerlos en cualquier momento.
US0008 Como usuario Quiero poder calificar las reseñas o obras que estén en la plataforma. Con el fin de Dar una opinión rápida sobre los trabajos de otros usuarios
US0009 Como usuario Quiero poder contestar los comentarios que le hagan a mis reseñas. Con el fin de opinar sobre los comentarios recibidos.
US0010 Como usuario Quiero poder borrar o editar mis reseñas cuando quiera. Con el fin de controlar su contenido.
US0011 Como usuario Quiero poder ver el nivel de calificación de cada una de las obras que estén en la plataforma. Con el fin de tener una medición rápida de calidad.



Tareas Técnicas

  • Incluir mecanismo de autenticación para el acceso al sitio en web (plataforma) por medio de usuario y contraseña.
  • Diseñar e implementar pruebas unitarias y de integración.
  • Establecer diferentes roles y permisos de usuario.
  • Implementar del modelo de usuario en la base de datos.
  • Crear pestaña funcional de Sign Up(Registro).
  • Crear pestaña funcional de Login(entrada).
  • Crear pestaña funcional de Log Out(Salida).

El sistema de software debe cumplir ciertos requerimientos, los cuales están relacionados con el manejo de la lógica y de los datos, la eficiencia de la aplicación y la usabilidad de la misma:


El sistema debe aplicar un algoritmo de ciframiento asimétrico sobre la contraseña de los usuarios.
Esto con el objetivo de proteger los datos del usuario al impedir que cualquiera pueda obtener su contraseña. Las contraseñas de los usuarios son en particular importantes debido a que la mayoría de las personas usan una clave común para múltiples plataformas.
